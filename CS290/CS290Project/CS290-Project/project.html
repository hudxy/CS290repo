  • Zombie Mansion Survival


    A C++ text based game that allows the player to move through a series of rooms in a abandoned mansion that has a zombie. The player needs to find and interact with items in the room to win the game. Constrictions include: player has finite amount of turns, limited inventory space, and must have specific items to complete the game.

  • Hackathon Hangman


    A C++ Hangman game that was submitted to Beaver Hacks Winter 2018 Hackathon. Game is allows for a dynamic number of words to be added to play hangman. User can add words to a given text file and the one word will be randomly selected by the program to play hangman.

  • Fantasy Battle Arena


    A C++ text game that allows the user to pick two fantasy characters and battle them. Most characters have special abilities that affect gameplay of each round of battle. Uses a linked-list structure that allows for a tournament style battle.

  • Zoo Tycoon


    A C++ text game that creates a simple game similar to the popular Zoo Tycoon. It allows the user to run a small zoo business and manage expenses, return profits, and buy new animals for the zoo.
